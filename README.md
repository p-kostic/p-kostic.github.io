# p-kostic.github.io

Template from [cotes2020/jekyll-theme-chirpy](https://github.com/cotes2020/jekyll-theme-chirpy). Below is some of the template's original documentation to get you started for when you want to run it locally.

## Running locally
If you would like to run or build the project on your local machine, please follow the [Jekyll Docs](https://jekyllrb.com/docs/installation/) to complete the installation of `Ruby`, `RubyGems`, `Jekyll` and `Bundler`.

Make sure to install all the required packages first
```terminal
bundle install
```

Now that everything is ready, you can run

```terminal
bundle exec jekyll s
```
Then open a browser window and visit http://localhost:4000
